Simple Minetest Mod Maker

This tool allows non-programmers to make simple mods for Minetest using an easy graphical interface. Modding in Minetest is quite easy. This editor cannot be used to create mobs, or to make that mod in your dreams that fixes everything in Minetest. If you want to create anything more than a decorative block, you need to learn how to mod Minetest properly, using Lua.

Easy to Use

This tool gives you simple instructions on each page. The tool is designed for those with little knowledge of the Minetest internals, and so is newbie friendly.

Saving and Opening projects

To open a project, drag and drop the file onto the webpage. To save a project, click save on the main page.
